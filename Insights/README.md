## Getting Started

### Prerequisites

make

go v1.18+

go-libp2p

go-libp2p-kad-dht

### Installation

```bash
# Clone the repository
git clone https://github.com/kahshiuhtang/Insights.git

# Change into the project directory
cd Insights

# Build the project
make build

# Run the project
make run
```
